> 版本: v1.0 · 更新日期: 2026-04-21  
> 状态图例: ✅ 已完成 · 🟡 部分完成（API/数据层已实现，UI 未实现）· ❌ 未实现

---

## 概述

KMPExamSystem 是一套基于 Kotlin Multiplatform（KMP）的跨平台在线考试系统，支持 Android 与桌面端。系统设有**管理员（ADMIN）**、**教师（TEACHER）**、**学生（STUDENT）**三种角色，通过 JWT 鉴权与 RBAC 实现功能隔离。

## 七、优先级建议（未完成功能）

### 高优先级（影响核心业务闭环）
1. **❌ 考试组卷 UI** — 教师在创建考试后，无法通过 UI 向考试中添加题目，导致考试无法真正出卷
2. **❌ 评分界面（主观题）** — 教师无法在 App 内查看提交并评分，AI 评分功能虽已有 API 但无 UI
3. **❌ 成绩详情页面** — 学生提交后只能看总分，无法看到各题解析和得分
4. **❌ 学生退选课程** — 选课后无法退出

### 中优先级（影响完整体验）
5. **❌ 题库管理 UI** — `QuestionBankApi` + Repository 完整，需补充题库管理页面
6. **❌ 个人信息修改/改密** — 学生无法自主维护个人信息
7. **❌ 管理员统计仪表盘** — 系统总览、课程/考试统计 API 已完备，需专属首页
8. **🟡 AI 配置管理界面** — 管理员无法在 UI 上查看和修改 AI 配置

### 低优先级（锦上添花）
9. **❌ 数据图表可视化** — 目前统计数据均为纯文本展示
10. **❌ 成绩导出** — 教师导出考试成绩为 Excel
11. **❌ 题目导入/导出 UI** — API 已完整，需文件选取组件支持
12. **❌ 系统广播通知** — 管理员向全体用户推送通知的界面
13. **🟡 监考事件前端检测** — 切屏/失焦时自动上报
