# KMP考试系统 - UI功能总结（设计师版）

## 项目概述
Kotlin Multiplatform考试系统，支持Android、iOS、Desktop三端。Material Design 3设计语言。

---

## 一、页面清单与核心功能

### 1. 认证模块

| 页面 | 核心功能 | 关键元素 |
|------|----------|----------|
| **登录页** | 用户名+密码登录 | 输入框(2)、登录按钮、密码显隐切换、注册链接 |
| **注册页** | 新用户注册 | 输入框(4)、角色选择(学生/教师)、注册按钮、登录链接 |

### 2. 学生模块

| 页面 | 核心功能 | 关键元素 |
|------|----------|----------|
| **首页** | 快速入口+通知 | 问候语、未读通知角标、通知卡片(3条)、即将考试列表 |
| **课程中心** | 选课/退课 | Tab切换(全部/已选)、课程卡片、选课按钮、退课按钮、课程考试弹窗 |
| **考试列表** | 查看/参加考试 | Tab切换(可参加/已结束)、考试卡片(标题/时长/满分/题数)、开始考试按钮 |
| **考试答题** | 答题+交卷 | 题目卡片、选项(单选/多选/判断/简答)、倒计时、交卷按钮、切屏警告 |
| **成绩历史** | 查看成绩统计 | 统计卡片(参加数/平均分/最高分/完成率)、成绩记录列表 |
| **成绩详情** | 查看批改详情 | 逐题展示(题目/学生答案/正确答案/解析/批语)、对错标记 |
| **通知中心** | 查看/管理通知 | 通知列表、滑动删除、标记已读、全部已读、加载更多 |
| **个人中心** | 账户管理 | 用户信息卡、菜单列表(考试历史/通知/密码/帮助)、退出登录 |

### 3. 教师模块

| 页面 | 核心功能 | 关键元素 |
|------|----------|----------|
| **考试管理** | 管理考试 | Tab切换(草稿/进行中/已结束)、考试卡片、新建按钮、批量操作 |
| **组卷页面** | 选题组卷 | 题目列表、分数统计、智能组卷弹窗(题型规则配置) |
| **答卷列表** | 查看学生答卷 | 学生卡片(姓名/切屏次数/分数/状态)、批改入口 |
| **批改试卷** | 批改主观题 | 题目卡片(参考答案/学生答案)、得分输入、批语输入、AI判分按钮 |

### 4. 管理员模块

| 页面 | 核心功能 | 关键元素 |
|------|----------|----------|
| **管理仪表盘** | 系统概览 | 8项统计卡片(用户/学生/教师/管理员/课程/考试/题目/提交)、课程通过率图表 |
| **用户管理** | 管理用户 | 搜索栏、角色筛选、用户卡片列表、新建按钮、批量删除、分页 |
| **题库管理** | 管理题库和题目 | 题库列表(左栏)、题目列表(右栏)、类型/难度筛选、Excel导入、新建按钮 |
| **AI配置** | 管理AI评分配置 | 配置项列表(key-value)、编辑按钮、保存按钮 |

## 三、关键交互模式

### 1. 导航结构
- **学生**：底部导航栏(首页/课程/考试/我的)
- **教师**：底部导航栏(首页/课程/考试管理/我的)
- **管理员**：底部导航栏(首页/用户/课程/题库/考试管理/系统/我的)

### 2. 表单交互
- **输入框**：Material Design 3 OutlinedTextField
- **选择器**：FilterChip(角色/题型/难度)、DropdownMenu(课程/题库)
- **日期时间**：Material3 DatePicker + TimePicker
- **按钮**：主按钮(FilledButton)、次按钮(OutlinedButton)、图标按钮

### 3. 反馈机制
- **成功/失败**：Snackbar提示
- **确认操作**：AlertDialog弹窗
- **加载状态**：CircularProgressIndicator
- **空状态**：图标+文字提示

### 4. 特殊交互
- **滑动删除**：通知列表支持左滑删除
- **批量选择**：用户管理/考试管理支持批量模式
- **实时统计**：组卷页面实时计算总分
- **切屏检测**：考试页面检测窗口焦点，显示警告

## 五、设计规范提示

### 颜色
- 主色：`#006495`(浅色模式) / `#8DCDFF`(深色模式)
- 使用Material Design 3动态取色系统

### 布局
- 桌面端：内容区限宽(`max-width: 800-1000dp`)
- 移动端：全宽布局
- 卡片间距：`8-16dp`
- 内边距：`16-24dp`

### 组件
- 卡片：`Card` + `elevation: 1-2dp`
- 按钮：`FilledButton`(主操作)、`OutlinedButton`(次操作)
- 输入框：`OutlinedTextField`
- 图标：Material Symbols

### 响应式
- 支持手机、平板、桌面三端
- 桌面端使用侧边栏导航
- 移动端使用底部导航栏

## 六、特殊场景

### 1. 考试模式
- 进入考试后锁定导航，防止误操作
- 倒计时显示，最后10分钟变红
- 切屏超过阈值强制交卷
- 交卷前确认对话框

### 2. 批量操作
- 用户管理：批量删除用户
- 考试管理：批量删除草稿考试
- 底部显示操作栏

### 3. AI辅助
- 批改试卷时可调用AI判分
- 显示AI建议分数和评语
- 教师可修改AI建议
